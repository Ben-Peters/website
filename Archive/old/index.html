Welcome

Welcome to the WPI Lens and Lights club website. We are a Class 3 organization that provides professional quality lighting, sound, and projection services for on and off campus events.

If you would like to submit a work order for an event please fill out a work order form. You can find the current prices in our Price List. If you have any questions regarding submitting a work order please email lnl@wpi.edu and we will respond to you as quickly as possible.

Otherwise, feel free to browse around the site to learn more about the organization.
